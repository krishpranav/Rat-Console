# Rat-Console
# A simple rat tool build on python

# Steps to run this on linux:
- git clone https://github.com/krishpranav/Rat-Console
- cd Rat-Console
- sudo chmod +x *
- bash install.sh
- python3 console.py 

# Steps to run this tool on mac
- git clone https://github.com/krishpranav/Rat-Console
- cd Rat-Console
- sudo chmod +x *
- bash install.sh
- python3 console.py
